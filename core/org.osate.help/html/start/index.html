Getting started

OSATE is a complete modeling environment that provides support for describing AADL models. It is composed of many features to model and analyse systems. Also, user can build their own tools based on OSATE and extend its initial capability.

The following manual provides an overview of the basic features of OSATE to :

  • create a project
  • write a first model
  • navigating in the AADL Text Editor
  • instantiate your model
