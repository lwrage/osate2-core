2.3 Verify the Installation
Run Eclipse. Go to Help | About Eclipse Platform. In addition to the icon for the Eclipse features () you should see an icon for the OSATE features (). Click on the Eclipse icon to view the list of Eclipse features. The list should include several EMF-related features, such as Eclipse Modeling Framework (EMF), and EMF Service Data Objects (SDO). Click on Ok. Click on the OSATE icon to view the list of OSATE features. You should see a single feature: Aadl OSATE.
