8. The AADL Graphical Editor
The graphical editor is provided by the TOPCASED initiative (www.topcased.org).
OSATE now supports editing AADL models in the text editor, the object editor, and in the graphical editor. Any changes made to an AADL model in one editor are synchronized with the other editor when a save is performed. Each editor allows you to go to the location of the current selection in either of the other two editors.  When double clicking on Markers in the Problem View, the user is placed at the marker location in the editor that responds to the file with the marker.  The text editor responds to aadl files, the object editor responds to aaxl files, and the graphical editor responds to aaxldi files.
