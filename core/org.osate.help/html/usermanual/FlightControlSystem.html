11.8 Flight Control System
A flight control system example used by ENST in a training exercise.   It is intended to be used as a tutorial for developing a small yet functional AADL system that models an aircraft's flight control system.  Although it is a simplistic model, an effort was made to incorporate almost all constructs of AADL into it.
